# 抽題轉盤

不重複抽題的靜態網頁應用，適合團康活動、課堂互動、派對遊戲等場景使用。

## 功能

- **隨機抽題** — 點擊按鈕或按空白鍵抽取題目，附帶轉盤動畫與彩紙特效
- **不重複抽取** — 抽過的題目不會再出現，畫面顯示剩餘題數
- **抽題紀錄** — 右側欄即時顯示所有已抽題目，最新的排在最上方
- **紀錄保留** — 重新整理頁面時可選擇繼續上次進度或重新開始（使用 localStorage）
- **RWD** — 支援桌面與手機瀏覽

## 使用方式

直接用瀏覽器開啟 `index.html` 即可使用，不需要任何伺服器或建置工具。

## 修改題目

編輯 `index.html` 中的 `DEFAULT_TOPICS` 陣列：

```javascript
const DEFAULT_TOPICS = [
  '題目一',
  '題目二',
  '題目三',
  // ...
];
```

## 技術

純 HTML / CSS / JavaScript，無任何外部框架依賴。
